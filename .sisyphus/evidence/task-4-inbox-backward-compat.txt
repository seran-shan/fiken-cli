List documents in the EHF (electronic invoice) inbox.

Usage:
  fiken inbox [flags]
  fiken inbox [command]

Available Commands:
  upload      Upload a document to the inbox

Flags:
  -h, --help            help for inbox
      --status string   Filter by status (pending, processed)

Global Flags:
      --company string           Company slug (auto-detected if only one)
      --json                     Output as JSON
      --keyring-backend string   Keyring backend: auto, secret-service, keychain, wincred, pass, file (default "auto")
      --no-input                 Non-interactive mode

Use "fiken inbox [command] --help" for more information about a command.
